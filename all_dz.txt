
Задача1 
Создать форму с двумя полями ввода (input) и кнопкой (button)
При нажатии на кнопку текст, введенный в первом поле ввода, должен появиться во втором поле.

Задача2
Создать форму для ввода: фамилии, имени и отчества
При вводе значений в поля ввода необходимо выводить на форме ФИО автоматически в формате: 
"Фамилия Имя Отчество" 

Задача3
Создать форму для авторизации в системе 
(ввод логина и пароля):  поле ввода  "Логин", поле ввода "Пароль", и кнопка "Вход" 
При нажатии на кнопку "Вход" необходимо вывести 
сообщение об ошибке, если логин и пароль не заполнен 
(формат вывода сообщения на ваше усмотрение). 
Если логин и пароль заполнен - вывести сообщение о успешной авторизации в системе в формате: 
"Выполнен успешный вход в систему (<введенный логин>)! "
Поля ввода и кнопка Вход в случае успешного входа должны быть "скрыты" (удалены из формы)

Задача4
Создать форму для вывода информации о компаниях в виде таблицы. 
Таблица должна содержать 3 колонки: Порядковый номер компании, Название компании, ИНН.
В качестве структуры для хранения данных в экземпляре Vue - использовать массив объектов (компаний). 
Заполнение массива компаний необходимо выполнить в хуке created() экземпляра Vue.

Задача5
Создать форму для конвертации из градусов фаренгейта в градусы цельсия.
Форма должна содержать поле ввода для ввода числового значения в градусах фаренгейта.
Формула: С = (F - 32) / 1.8, где F - значение в градусах фаренгейта.
Результат вычисления должен отображаться на форме после ввода данных и нажатия на кнопку "ОК"

Задача6
Доделать (доработать) стандартный пример - "список задач"
(https://jsfiddle.net/boilerplate/vue)
Дать возможность создавать новые задачи (поля название и статус)
После создания новой задачи она должна отображаться в списке задач. 
Вывести на форму (перед списком задач) количество всех задач и количество "выполненных" задач.

Задача7 
Сделать новый вариант "Задача5" без кнопки! 
Результат вычисления должен отображаться на экране "автоматически" при вводе значения (использовать "computed")
